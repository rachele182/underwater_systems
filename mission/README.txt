README

Per eseguire è sufficiente avviare la simulazione nel file TeamA_PAsqualo.slx. Sono stati forniti dei file .slx in versioni precedenti di matlab (2018b e 2019b).

I parametri relativi ad ogni blocco sono nel file init_NOME.m

I parametri relativi alla missione sono in missionA e in main_init.

Se si vuole avere un animazione della missione e` necessario eseguire, dopo la simulazione, lo script animazione.m.
Se si vuole avere un plot della missione e` necessario eseguire, dopo la simulazione, lo script plot2D.m.
Sia per lo script di animazione sia per quello di plot, si puo` caricare uno dei file.mat nella cartella plot_animazione. Ciascuno di questi, infatti, e` l'output di una simulazione.
